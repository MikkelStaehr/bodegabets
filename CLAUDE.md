## Design System
Always read and follow design-system/MASTER.md before writing any UI code.
Never use inline hex values — use CSS variables from globals.css.
Never use rounded-lg or rounded-2xl — use rounded-sm for all components, rounded-full for pills and avatars only.
Always use the spacing scale defined in MASTER.md.
Also read and follow the UI/UX guidelines in .claude/skills/ui-ux-pro-max/SKILL.md before writing any UI code.
